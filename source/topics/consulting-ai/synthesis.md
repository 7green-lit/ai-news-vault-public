## 2. 2023〜2025 までの前史(モデル知識ベース)

### 2.1 「コンサル × AI」の出発点(2023)
- ChatGPT(2022-11)、GPT-4(2023-03)後、**Big4 と戦略系は「AI 部隊」を急速に新設**
  - **McKinsey QuantumBlack**(2015〜、AI/データ部隊。GenAI 時代に拡大)
  - **BCG X**、**Bain Vector**、**Deloitte AI Institute**、**PwC AI Lab**、**EY.ai**、**KPMG Trusted AI**
  - **Accenture が $3B 投資**を宣言(2023)、**1.5万人を AI 育成**
- **OpenAI / Microsoft / Google が直接コンサルと組まず、エコシステム経由で提供**するモデルが先行
- 国内では**NEC cotomi、富士通 Takane、ELYZA、PFN、Sakana** 等の独自 LLM 開発が先行、コンサル業界自体は「PoC 受託」が中心

### 2.2 「PoC 大量生産期」(2024)
- **MIT Project NANDA**「The GenAI Divide」が**95%の企業 GenAI 投資はリターンを生まない**と報告(2025)
- Big4 + 国内 SI 主導で**大量 PoC**が乱立、**「PoC から本番に進まない」現象**が常態化
- McKinsey, BCG が**GenAI ROI 調査**を四半期で発表
- アクセンチュアが SAP と**「ADVANCE」(AI ネイティブ業務設計パッケージ)**の原型開発(2024)

### 2.3 「実装フェーズ前夜」(2025)
- **Anthropic Cowork、Claude Code、OpenAI Codex、Cursor**等の**AI 駆動開発ツール**普及で、**コンサル成果物の「実装フェーズ」**が AI に侵食され始める
- **Palantir Forward Deployed Engineer**(FDE)モデルが**「AI ベンダー直営の実装支援」**として注目
- 国内では**富士ソフト**が「人月商売脱却」検討開始、**TIS / SCSK / BIPROGY / CTC**が**AI 戦略の見直し**を表明

### 3.2 6つの主要動向

#### A. AI ベンダーの直接侵入(最大の構造変化)
- **2026-05-12 OpenAI Deployment Company 設立 + $4B 調達 + Tomoro 買収** で**FDE モデルが汎用 AI 業界に到来**
- Anthropic は NEC × Anthropic(2026-05-11)、Goldman/Blackstone $1.5B(2026-05-05)、AWS ネイティブ提供(同日)で**3軸の業界侵入**
- Sakana × SMBC、Sakana × ASBC、Sakana × UBE は**特化型コンサルとして既に侵入完了**

#### B. Big4 の AI 提携先行 / 受身化
- **PwC × OpenAI**(2026-05-05) が Big4 最先行
- Deloitte / EY / KPMG の追随報道は 2026-05 時点で限定的(過去:Deloitte が Anthropic 提携、EY が OpenAI 提携の前例あり)
- Big4 の差別化軸:**監査 + 規制対応 + 内部統制** — EU AI Act(2024-08)、日本 AI 法律(2026-05-11)、Anthropic Claude メキシコ攻撃事件(2026-05-08)、AI 監査需要が新規開拓余地

#### C. 戦略系の AI ROI 助言継続
- McKinsey / BCG / Bain は**「戦略策定」+「変革管理」**で残る可能性が高い
- ただし、**ChatGPT 5.5 Pro の数学者(Gowers)体験記**(2026-05-09)が示すように、**戦略策定の中核部分(構造化思考、フレームワーク選択)も AI が部分置換**可能
- 2026 年度内に **GenAI 戦略マスタークラス + 経営層向け教育**で**短期収益**を上げるパターンが続く

#### D. アクセンチュア型(規模 + 業界 + 実装)の堅調
- **アクセンチュア × SAP ADVANCE**(2026-04-30 国内本格展開)、**アクセンチュア × NSK**(2026-05-03)、**アクセンチュア × Anthropic 提携**(2025〜継続中、海外メイン)で**「業界 × 規模 × 実装」**で差別化
- IBM Consulting(IBM Sovereign Core 2026-05-11 と連動)、Capgemini、TCS / Infosys が**同モデル**で追走

#### E. 国内 SI / シンクタンクの構造変革
- **富士ソフト人月脱却**(2026-04-30)、**日立 GlobalLogic 統合**(2026-05-11)、**富士通脱却**(2026-05-12 英 HMRC + 郵便局 → AWS)、**NEC × Anthropic**(2026-05-11)、**ELYZA × KDDI**(2026-05-11)、**三菱UFJリサーチ「Gov Sales」**(2026-05-12)、**NRI「AI と組織の未来」**(2026-05-12)
- **3つの戦略分岐**:
  1. **人月脱却 + 成果報酬**(富士ソフト)
  2. **規模拡大 + 海外 SI 取込**(日立 GlobalLogic、富士通の苦戦反転狙い)
  3. **米AI 大手提携 + 国内市場特化**(NEC × Anthropic、ELYZA × KDDI)
- シンクタンクは**「公共 × AI」**で AI ネイティブ競合が少ない領域に集中(Gov Sales、NRI 調査)

#### F. AI ネイティブの脅威加速
- **Cognition AI(Devin)日本法人**(2026-04-29)で**自律 AI エンジニア時間貸し**モデルが国内上陸
- **ダイヤモンド「AI ネイティブスタートアップの脅威」**(2026-05-11)が**業界全体への構造的危機**を整理
- Sierra AI、Decart.ai、Lovable、Cursor、Replit、Continue.dev 等が**コンサル中間成果物を直接生成**

## 4. 横断視点 — コンサル業界 5層分化仮説

2026-05 時点で**コンサル業界が以下の5層に分化しつつある**:

| 層 | 担い手 | 競争優位 | リスク |
|---|---|---|---|
| **戦略立案** | McKinsey/BCG/Bain | 経営層関係 + 思考フレーム + 業界知識 | 戦略策定自体の AI 補助で**短時間化**、上位フィー圧迫 |
| **変革管理 / 組織** | NRI / アビーム / Big4 ストラテジー部隊 | 人材育成 + 文化変革 + 内部統制 | AI ネイティブ企業は**変革管理を内製化**、外部不要 |
| **実装 / SI(業界特化)** | Accenture / IBM Consulting / Capgemini / NEC / 富士通 / 日立 | 業界知識 + 規模 + 既存大型契約 | **AI ベンダー直営 FDE モデル**で**コモディティ業務を侵食**される |
| **公共 / 規制** | Big4 監査部門 / NRI / 三菱UFJリサーチ / 三菱総研 | 規制対応 + 公共調達 + 監査独立性 | EU AI Act / 日本 AI 法律で**規制対応が一時的に主要収入**になる過渡期 |
| **AI ネイティブ侵入** | OpenAI Deployment Co + Tomoro / Sierra / Cognition / Sakana / Anthropic Solutions | AI 性能 + スピード + 低コスト | 「業界知識」「規模」「人脈」が欠ける、大型クライアントには限界 |

## 5. 国内 vs 海外 ── 構造比較

| 観点 | 海外(米英) | 日本 |
|---|---|---|
| 主導 | Big4 + 戦略系(McKinsey 等)+ Palantir 型 FDE | アクセンチュア(外資) + 国内 SI + シンクタンク |
| 人月モデル | 既に成果報酬化 / プロジェクト型化 | **人月モデルが依然主流、富士ソフトが先行脱却(2026-04-30)** |
| AI ベンダー直接侵入 | OpenAI Deployment Co(2026-05-12)で本格化 | NEC × Anthropic(2026-05-11) — 国内 SI 経由の提携で間接的 |
| 公共 / シンクタンク | 戦略系が公共助言、Big4 が監査・規制 | NRI、三菱総研、三菱UFJリサーチ、大和総研、日本総研 + 政府系研究機関 |

3. **米AI ベンダーが国内 SI 経由で攻略**(NEC × Anthropic、ELYZA × KDDI、Cognition AI 日本法人)するパターンが定着しつつある
4. **公共セクター案件**(霞が関 18万人 AI、Gov Sales、一関市 RAG)が**コンサル / シンクタンクの新セグメント**として成長

## 6. 見通し

### 6.1 短期(2026 Q3-Q4)
- **Deloitte / EY / KPMG が OpenAI / Anthropic / Google と提携発表**(Big4 完了)
- **アクセンチュアと OpenAI / Anthropic の直接競合化**が露呈、価格圧力が始まる
- **国内 SI(TIS、SCSK、BIPROGY、CTC)が AI 戦略を本格表明**(富士ソフト型 or 日立型を選ぶ)
- **AI 監査** が Big4 にとって新規収入軸として立ち上がる(日本 AI 法律施行で2026-2027 ピーク)
- **OpenAI Deployment Co の初期成果が公開**、Tomoro 統合の現実性が試される

### 6.2 中期(2027)
- **戦略系(McKinsey/BCG/Bain)の上位フィー圧迫**(GenAI 戦略コンサルの単価半減)
- **アクセンチュアの「業界 × 規模」優位が薄れる**、AI ネイティブ FDE モデルがコモディティ業務を席巻
- **シンクタンク(NRI、三菱UFJリサーチ等)が「実プロダクト + 政策助言」のハイブリッド**へ転換、AI ツール内製化
- **国内 SI の M&A 加速**(日立 GlobalLogic 統合の続発、富士通 SI 部門切り出し説の浮上)
- Cognition / Sierra / Tomoro / Sakana 等の**AI ネイティブが Tier 2 〜 Tier 3 のコンサル領域を吸収**

### 6.3 長期(2028+)
- **「コンサル業」自体が4つに分裂**:
  1. **上位戦略助言**(経営層 × 戦略系、最小限の人数で高単価)
  2. **規制 / 監査 / 公共**(Big4 + シンクタンク、規制対応の独占)
  3. **AI ベンダー直営 FDE**(OpenAI / Anthropic / Sakana / Cognition 直営)
  4. **国内ローカル + 規制対応 SI**(NEC、富士通、日立、富士ソフト等が縮小均衡)
- 中堅 SI(TIS、SCSK、CTC、BIPROGY 等)の**多くが買収 / 統合 / 縮小**
- **シンクタンク × AI ツールの「政策実装プラットフォーム」**が新カテゴリとして確立
